# mall

## 目录结构
- src
  - assets 需要的图片和通用的css配置
  - common 通用的js函数
  - components 组件
    - common 跨项目通用组件
    - content 本项目通用组件
  - network 请求封装和封装一些数据
  - router 路由相关配置
  - store vuex相关配置
  - views 页面
    - home 主页
    - category 分类页面
    - cart 购物车页面
    - profile 个人主页
    - detail 商品详情页
